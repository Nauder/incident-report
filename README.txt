Para inicialização correta do projeto é necessário a instalação de maven e postgresql no dispositivo.

Também é necessária a criação de um banco de dados no postgre com o nome "distributed", role com o nome "nauder" e senha "505450"
ou edição do arquivo "DBConnection.java" para utilizar os credenciais da máquina.

O arquivo "database.sql" contém a sql para criação das tabelas, após o banco já ter sido criado.

Ferramentas utilizadas:
Java 20.0.1
JavaFX 20
PostgreSQL 15.3
Maven 4.0.0
